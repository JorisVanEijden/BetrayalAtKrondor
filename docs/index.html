ReverseBak Project

Rediscovering the magic of Betrayal at Krondor

About the Project

ReverseBak is an ambitious endeavor to reverse engineer the classic MS-DOS game Betrayal at Krondor. Our goal is to understand the game's data formats and mechanics to eventually create a highly moddable version in Unity.

Current Components

ResourceExtractor

  • Converts original game data files to .NET objects
  • Exports data as JSON/CSV for analysis
  • Development tool with command line interface

BetrayalAtKrondor

  • Works with Spice86 DOS emulator
  • Provides C# overrides for original game functions
  • Enables debugging, logging and investigation of game behavior
  • Replaces low-level DOS file handling for better control
